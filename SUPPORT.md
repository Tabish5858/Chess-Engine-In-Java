# Support

## Getting Help

- Usage and setup questions: open a GitHub Discussion or Issue
- Bug reports: use the bug issue template
- Feature ideas: use the feature request template
- Security issues: follow [SECURITY.md](SECURITY.md)

## Before Opening an Issue

- Confirm your JDK version is 17+
- Ensure dependencies in [JChess](JChess) are present
- Share stack traces and exact reproduction steps
